## 🛠️ Installation

### 1. Clone Repo

```
git clone https://github.com/your-repo/swp-scaffolding.git
cd swp-scaffolding
```

### 2. Install Dependencies (if applicable)

```
npm install
```

### 3. Run Locally

```
npm run dev
```

### 4. Build for Production

```
npm run build
```
